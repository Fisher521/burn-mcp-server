Burn MCP Server

MCP Streamable HTTP Read less, absorb more.

This is the remote HTTP endpoint for Burn's Model Context Protocol server. 26 tools to let Claude / Cursor / Windsurf search, triage, and analyze your saved reading.

Connect

Paste this into your MCP client (Claude Desktop, Cursor, Windsurf, etc.):

{
  "mcpServers": {
    "burn": {
      "url": "https://mcp.burn451.cloud/mcp",
      "headers": {
        "Authorization": "Bearer <your-token>"
      }
    }
  }
}

Get your token: burn451.cloud → Settings → MCP Server → Copy Access Token
